# Olitun Terms Of Use

Last updated: May 5, 2026

Olitun is provided as an educational app for learning Ol Chiki and related Santali learning content. By using the app, you agree to use it respectfully and only for lawful learning, teaching, and administration.

## Accounts

Users are responsible for the accounts they create and for keeping sign-in access secure. Admin access is limited to authorized team members and must only be used to manage appropriate learning content.

## Content

Learning content, media, translations, and examples should not be copied, uploaded, or shared unless you have the rights to use them. Admins should verify rights before publishing media or lesson content.

## Availability

Features may change as the app improves. Offline features can continue to work with local data, but cloud sync, account features, media, and translation require network access.

## Prohibited Use

Do not abuse the app, bypass access controls, overload translation services, upload unsafe content, or use the platform in a way that harms other users or the service.
